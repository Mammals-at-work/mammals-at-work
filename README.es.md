## Stack Tecnológico

- **[Svelte 5](https://svelte.dev/)** — Framework de UI
- **[Vite](https://vitejs.dev/)** — Herramienta de construcción
- **[Tailwind CSS v4](https://tailwindcss.com/)** — Estilos
- **[Lucide](https://lucide.dev/)** — Iconos
- **[Gemini API](https://ai.google.dev/)** — Capacidades de IA

## Primeros Pasos

**Requisitos previos:** [Node.js](https://nodejs.org/) (v18+)

```bash
# Clonar el repositorio
git clone https://github.com/Mammals-at-work/mammals-at-work.git
cd mammals-at-work

# Instalar dependencias
npm install

# Configurar el entorno
cp .env.example .env.local
# Edita .env.local y añade tu GEMINI_API_KEY

# Iniciar el servidor de desarrollo
npm run dev
```

La aplicación estará disponible en `http://localhost:3000`.

### Scripts Disponibles

| Comando | Descripción |
|---------|-------------|
| `npm run dev` | Inicia el servidor de desarrollo en el puerto 3000 |
| `npm run build` | Compila para producción |
| `npm run preview` | Previsualiza la compilación de producción |
| `npm run lint` | Verificación de tipos con TypeScript |
| `npm run clean` | Elimina la carpeta `dist/` |
