## Primers Passos

**Requisits previs:** [Node.js](https://nodejs.org/) (v18+)

```bash
# Clonar el repositori
git clone https://github.com/Mammals-at-work/mammals-at-work.git
cd mammals-at-work

# Instal·lar dependències
npm install

# Configurar l'entorn
cp .env.example .env.local
# Edita .env.local i afegeix la teva GEMINI_API_KEY

# Iniciar el servidor de desenvolupament
npm run dev
```

L'aplicació estarà disponible a `http://localhost:3000`.

### Scripts Disponibles

| Comanda | Descripció |
|---------|------------|
| `npm run dev` | Inicia el servidor de desenvolupament al port 3000 |
| `npm run build` | Compila per a producció |
| `npm run preview` | Previsualitza la compilació de producció |
| `npm run lint` | Verificació de tipus amb TypeScript |
| `npm run clean` | Elimina la carpeta `dist/` |
