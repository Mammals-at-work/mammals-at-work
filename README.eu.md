## Lehen Pausoak

**Aurrebaldintzak:** [Node.js](https://nodejs.org/) (v18+)

```bash
# Biltegia klonatu
git clone https://github.com/Mammals-at-work/mammals-at-work.git
cd mammals-at-work

# Mendekotasunak instalatu
npm install

# Ingurunea konfiguratu
cp .env.example .env.local
# Editatu .env.local eta gehitu zure GEMINI_API_KEY

# Garapen zerbitzaria abiarazi
npm run dev
```

Aplikazioa `http://localhost:3000` helbidean egongo da erabilgarri.

### Script Erabilgarriak

| Komandoa | Deskribapena |
|----------|--------------|
| `npm run dev` | Garapen zerbitzaria abiarazi 3000 portuan |
| `npm run build` | Produkziorako konpilatu |
| `npm run preview` | Produkzio konpilazioa aurreikusi |
| `npm run lint` | TypeScript-ekin mota egiaztapena |
| `npm run clean` | `dist/` karpeta ezabatu |
