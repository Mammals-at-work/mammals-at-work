## Primeiros Pasos

**Requisitos previos:** [Node.js](https://nodejs.org/) (v18+)

```bash
# Clonar o repositorio
git clone https://github.com/Mammals-at-work/mammals-at-work.git
cd mammals-at-work

# Instalar dependencias
npm install

# Configurar o contorno
cp .env.example .env.local
# Edita .env.local e engade a túa GEMINI_API_KEY

# Iniciar o servidor de desenvolvemento
npm run dev
```

A aplicación estará dispoñible en `http://localhost:3000`.

### Scripts Dispoñibles

| Comando | Descrición |
|---------|------------|
| `npm run dev` | Inicia o servidor de desenvolvemento no porto 3000 |
| `npm run build` | Compila para produción |
| `npm run preview` | Previsualiza a compilación de produción |
| `npm run lint` | Verificación de tipos con TypeScript |
| `npm run clean` | Elimina o cartafol `dist/` |
