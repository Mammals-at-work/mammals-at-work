- **[Gemini API](https://ai.google.dev/)** — AI機能

## はじめに

**前提条件:** [Node.js](https://nodejs.org/) (v18+)

```bash
# リポジトリをクローン
git clone https://github.com/Mammals-at-work/mammals-at-work.git
cd mammals-at-work

# 依存関係をインストール
npm install

# 環境を設定
cp .env.example .env.local
# .env.local を編集して GEMINI_API_KEY を追加

# 開発サーバーを起動
npm run dev
```

アプリは `http://localhost:3000` で実行されます。

### 利用可能なスクリプト

| コマンド | 説明 |
|---------|------|
| `npm run dev` | ポート3000で開発サーバーを起動 |
| `npm run build` | 本番用にビルド |
| `npm run preview` | 本番ビルドをプレビュー |
| `npm run lint` | TypeScriptで型チェック |
| `npm run clean` | `dist/` フォルダを削除 |
