## Primero Paso

**Requihito previo:** [Node.js](https://nodejs.org/) (v18+)

```bash
# Cloná er repositorio
git clone https://github.com/Mammals-at-work/mammals-at-work.git
cd mammals-at-work

# Instalá lah dependensia
npm install

# Configurá er entorno
cp .env.example .env.local
# Edita .env.local y añade tu GEMINI_API_KEY

# Arrancá er servidó de desarroyo
npm run dev
```

La aplicasión va a ehtá disponible en `http://localhost:3000`.

### Script Disponible

| Comando | Descripción |
|---------|-------------|
| `npm run dev` | Arranca er servidó de desarroyo en er puerto 3000 |
| `npm run build` | Compila pa produsión |
| `npm run preview` | Previsualisá la compilasión de produsión |
| `npm run lint` | Verificasión de tipo con TypeScript |
| `npm run clean` | Elimina la carpeta `dist/` |
